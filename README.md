# seoul_subway_info

클린 아키텍처를 적용한 서울시 지하철 정보 앱입니다. 현재 위치에서 가장 가까운 역의 실시간 지하철 도착 정보를 제공합니다.

- Geolocator를 이용해 현재 위치에서 가장 가까운 지하철 역을 찾아 실시간 도착 정보 제공 (서울시 공공 API 이용)
- Provider, Freezed, get_it & Injectable (DI) 적용
